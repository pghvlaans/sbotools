NAME

sbosnap - slackbuilds.org tree fetch and update command

SYNOPSIS

sbosnap [-h|-v]

sbosnap (fetch|update)

DESCRIPTION

sbosnap is used to download and update a local copy of the slackbuilds.org tree, without the *.tar.gz{,.asc} files. Note that sbocheck(1) also updates the tree and checks for updated and removed SlackBuilds. rsync is used for rsync repositories, and git is used for git repositories. sbotools3 defaults to git unless RSYNC_DEFAULT is TRUE. See sboconfig(1) or sbotools.conf(5).

OPTIONS

-h|--help

Show help information.

-v|--version

Show version information.

COMMANDS

fetch

Download a local copy of the slackbuilds.org tree. The copy will be downloaded to the path in the SBO_HOME setting (see sboconfig(1) and sbotools.conf(5)), /usr/sbo by default.

update

Update a previously fetched copy of the slackbuilds.org tree.

EXIT CODES

sbosnap can exit with the following codes:

0: all operations completed successfully.
1: a usage error occurred, such as running sbosnap with no command.
5: error downloading from the repository.
